    Express: Express es un framework de servidor web rápido, minimalista y flexible para Node.js. 
    Se utiliza para construir aplicaciones web y APIs.

    Express Handlebars: Handlebars es un motor de plantillas para generar HTML dinámico en el 
    lado del servidor. Express Handlebars es una implementación de Handlebars 
    diseñada específicamente para integrarse con Express.

    Morgan: Morgan es un middleware de registro de solicitudes HTTP para Node.js. 
    Se utiliza para registrar información sobre las solicitudes entrantes en el servidor, 
    como la URL solicitada, el método HTTP utilizado y el código de estado de la respuesta.

    Multer: Multer es un middleware para Express y Node.js que facilita la manipulación de 
    datos de formularios enviados a través de solicitudes HTTP multipart/form-data. 
    Se utiliza principalmente para manejar la carga de archivos en aplicaciones web.

    Mongoose: Mongoose es una biblioteca de modelado de objetos MongoDB (ODM) para Node.js. 
    Proporciona una solución sencilla y basada en esquemas para trabajar con MongoDB 
    desde Node.js, facilitando la definición de modelos, la validación de datos y 
    las operaciones de base de datos.

    Nodemon: Nodemon es una herramienta que ayuda en el desarrollo de aplicaciones Node.js 
    reiniciando automáticamente el servidor cuando se detectan cambios en los archivos 
    del proyecto. Es muy útil durante el desarrollo porque elimina la necesidad de 
    reiniciar manualmente el servidor después de cada cambio en el código.

    Dotenv: Dotenv es un módulo que carga las variables de entorno de un archivo .env en 
    el proceso de Node.js, lo que facilita la configuración de variables de entorno en el 
    entorno de desarrollo.

    Cross-env: Cross-env es una herramienta que permite definir variables de entorno de 
    manera consistente en diferentes sistemas operativos. Es especialmente útil para definir 
    variables de entorno en scripts de configuración y ejecutarlos en sistemas Windows y Unix.

    Cors: CORS (Cross-Origin Resource Sharing) es un mecanismo que permite que los recursos 
    de un servidor web sean accesibles desde un origen diferente al dominio donde se 
    encuentra el servidor. El paquete cors de Node.js proporciona un middleware que se 
    utiliza para habilitar CORS en una aplicación Express.

    Cloudinary: Cloudinary es un servicio de gestión de medios en la nube que proporciona 
    almacenamiento, manipulación y entrega de imágenes y videos optimizados. 
    El paquete cloudinary de Node.js es un cliente oficial que facilita la integración de 
    la plataforma Cloudinary en aplicaciones Node.js.
